# midnight-sample-fullstack-app

midnight-sample-fullstack-app

## How to work

You must do under `app` folder.

- install

  ```bash
  bun install
  ```

- build

  ```bash
  bun run build
  ```

- dev

  ```bash
  bun run dev
  ```

## Tech Stack

- Bun
- React
- TypeScript
- Lace Wallet
- MidNight
- PreProd
- Biome
