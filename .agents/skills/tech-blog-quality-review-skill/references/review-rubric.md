# Review Rubric (1-5)

## 1. Reader Fit
- 1: 誰向けか不明
- 3: 想定読者はあるが前提知識が曖昧
- 5: 対象読者・前提・到達点が冒頭で明確

## 2. Structure and Flow
- 1: 結論不明、話題が散乱
- 3: 見出し構造はあるが順序が弱い
- 5: 問題提起→結論→手順→検証→まとめが自然

## 3. Technical Accuracy
- 1: 誤り/曖昧表現が多い
- 3: 大筋正しいが条件や制約の記述不足
- 5: 正確で、バージョン・前提・例外条件まで明記

## 4. Reproducibility
- 1: 再現手順がない
- 3: 手順はあるが依存関係や環境差分で詰まる
- 5: コピペ可能な手順、前提、期待結果、失敗時対処が揃う

## 5. Readability and Completion Rate
- 1: 長文連続、要点不明
- 3: 可読だが冗長・重複がある
- 5: 1段落1論点、適切な改行/箇条書き、途中離脱しにくい

## 6. Practical Value
- 1: 読後に行動できない
- 3: 知識は得られるが次アクションが弱い
- 5: すぐ試せるチェックリスト/コード/判断基準がある

## Severity Labels
- Critical: 誤情報、危険な実装、結論欠落、再現不能
- Major: 構成破綻、読者不一致、根拠不足
- Minor: 表現改善、冗長削減、タイトル最適化
